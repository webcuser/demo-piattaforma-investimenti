# demo-piattaforma-investimenti

Simula una piattaforma fintech per la gestione di investimenti

## Setup Instructions

1. Install dependencies: `npm install`
2. Run the development server: `npm start`
3. Run the mock API server for testing: `node src/api/mockAPI.js`
4. Build the project: `npm run build`

## Overview

Questo progetto mira a simulare una piattaforma fintech avanzata per la gestione degli investimenti.

## Features

- Visualizzazione in tempo reale del portafoglio
- Analisi avanzata dell'andamento degli investimenti
- Grafici interattivi per una migliore comprensione dei dati
